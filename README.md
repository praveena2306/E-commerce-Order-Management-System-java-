# E-Commerce Order Management System

A Java-based console application to manage products, orders, and customers.

## Features
- Add products to inventory
- Create orders for customers
- View order history
- Update order status (Pending/Shipped/Delivered)
- Calculate total order price

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList and HashMap
- Console-based Menu Interface

## How to Run
1. Clone the repository: 
   `git clone https://github.com/praveena2306/ECommerce-Order-Management.git`
2. Navigate to the src folder: `cd src`
3. Compile Java files: `javac *.java`
4. Run main class: `java Main`

## Use Case
This project simulates a simple e-commerce system to track products, create customer orders, and update order status efficiently.
